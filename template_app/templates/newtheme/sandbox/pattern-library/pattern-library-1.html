Mixed/Nested Lists

description:

You can nest lists indefinitely.

example:

  • Morbi non dolor sed elit gravida elementum sit amet congue quam.
  • Aenean interdum felis a sollicitudin cursus.
    1. Morbi non dolor sed elit gravida elementum sit amet congue quam.
    2. Aenean interdum felis a sollicitudin cursus.
    3. Nam condimentum est ac ultrices congue.
  • Nam condimentum est ac ultrices congue.

code:

<ul>
  <li>...</li>
  <li>
    ...
    <ol>
      <li>...</li>
      <li>...</li>
      <li>...</li>
    </ol>
  </li>
  <li>...</li>
</ul>
