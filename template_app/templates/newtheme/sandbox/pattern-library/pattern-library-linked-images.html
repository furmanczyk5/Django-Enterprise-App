Code Structure

Image Grid

An image grid requires a <div class="image-grid-wrapper">, wrapping a <div class="image-grid">, containing several <div class="image-result-col"> elements, each one or more <div class="image-result"> elements. (see code below)

The <div class="image-grid"> element should also include addition classes to apply styling to the grid.

  • You must use one of the following classes to control the background of the overlaying info: info-overlay, info-overlay-partial
  • You must use one of the following classes to control the padding between images: lg-padding, md-padding, sm-padding, no-padding.

The <div class="image-result-col"> elements should also include addition classes to specify the width of each column and how they change with screen size. Use col-lg-4 col-md-6 col-sm-12 col-xs-12 (as in the code below), to go from 3 cols (large screen), to 2 cols (medium screen), to 1 col (small screen).

Do not wrap an image grid within <div class="layout-tracery">. You can do this, but it will add additional padding to the entire grid.

code:


<div class="image-grid-wrapper">
    <div class="image-grid info-overlay-partial md-padding">
        <div class="image-result-col col-lg-4 col-md-6 col-sm-12">
            <div class="image-result">...</div>
            <div class="image-result">...</div>
        </div>

        <div class="image-result-col col-lg-4 col-md-6 col-sm-12">
            <div class="image-result">...</div>
            <div class="image-result">...</div>
        </div>

        <div class="image-result-col col-lg-4 col-md-6 col-sm-12">
            <div class="image-result">...</div>
        </div>
    </div>
</div>

Image Result

Each <div class="image-result">...</div> element should contain one <a> element, which links to the location users are directed to when clicking through and wraps the image, and information (see code below for an example an image result and the classes to use)

Linked Images will not work unless wrapped within the image grid code above.

code:


<div class="image-result">
    <a href="/imagelibrary/details/9132257/">
        <img class="image-thumbnail" src="http://www.fortna.com/wp-content/uploads/o-MIDTOWN-ATLANTA-facebook.jpg">
        <div class="image-information">
            <div class="image-title">(1) Midtown Atlanta</div>
            <div class="image-subtitle">Atlanta, GA</div>
        </div>
    </a>
</div>

Examples

Sorting by column

description:

Place multiple <div class="image-result"> elements within a <div class="image-result-col"> element to organize and sort images into columns.

If you have images that are not all the same dimensions, you should use this method of sorting.

example:

Neighborhoods

Atlanta, GA
(2) Downtown Santa Ana
Santa Ana, CA

Sorting by row

description:

Individually wrap a <div class="image-result"> element within a <div class="image-result-col"> element to organize and sort images by row.

If you have images that are not all the same dimensions, the resulting grid will not have equal padding between rows.

example:

(1) Midtown Atlanta
Atlanta, GA
(2) Downtown Santa Ana
Santa Ana, CA
(3) Sherman Avenue
Coeur d’Alene, ID

Ketchum, ID
